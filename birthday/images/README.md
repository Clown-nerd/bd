# Photo Gallery Images

Add your photos to this folder with the following filenames:

- `photo1.jpg` — displayed in frame 1 (wide, top-left)
- `photo2.jpg` — displayed in frame 2 (portrait, top-right)
- `photo3.jpg` — displayed in frame 3 (portrait, middle-left)
- `photo4.jpg` — displayed in frame 4 (landscape, middle-right)
- `photo5.jpg` — displayed in frame 5 (wide, bottom)

The filenames must match exactly as listed (including the `.jpg` extension). If your photos are in a different format (e.g. `.png` or `.webp`), either convert them to JPEG or update the corresponding `src` attributes in `index.html` to use the correct extension.
